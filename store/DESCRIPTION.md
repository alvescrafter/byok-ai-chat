## Detailed description

```
Chat with any AI model directly from Chrome's side panel. BYOK AI Chat supports Bring Your Own Key — you provide the API key, you control your data, you choose the provider.

★ 6 AI PROVIDERS, ONE PLACE
• OpenAI (GPT-4o, GPT-4 Turbo, o1, and more)
• Anthropic (Claude 3.5 Sonnet, Claude 3 Opus, and more)
• Google Gemini (Gemini 1.5 Pro, Gemini 1.5 Flash, and more)
• Ollama — run models locally, no API key required
• LM Studio — local model server, no API key required
• Custom API — connect to any OpenAI-compatible endpoint (vLLM, Together AI, Groq, OpenRouter, etc.)

★ BUILT FOR POWER USERS
• Streaming responses with real-time token display
• Markdown rendering with syntax-highlighted code blocks
• Screenshot capture — grab the visible tab and send to vision models
• Page context — attach the current page's title, URL, selected text, and content
• Right-click any selected text → "Send to AI Chat"
• Conversation management with search, edit, regenerate, and export
• Multiple conversations with system prompt presets
• Dark and light themes
• Export to JSON, Markdown, or plain text
• Token estimation and context window management
• Voice input and text-to-speech output
• File attachments (images and text files)
• Conversation branching
• Keyboard shortcut (Ctrl+Shift+K) to open the side panel

★ YOUR DATA STAYS YOURS
• No analytics, no telemetry, no tracking
• API keys stored locally with chrome.storage
• Conversations stored locally on your device
• No data sent to any server except the AI provider you choose
• No developer middleman — direct browser-to-provider requests
• Open source: https://github.com/alvescrafter/byok-ai-chat

★ PERFECT FOR
• Developers who want AI assistance without leaving the browser
• Writers who need quick access to multiple AI models
• Researchers comparing responses across providers
• Privacy-conscious users who want to use their own API keys
• Anyone running local models via Ollama or LM Studio

★ GET STARTED IN SECONDS
1. Install the extension
2. Click the icon (or press Ctrl+Shift+K)
3. Open Settings, choose a provider, paste your API key
4. Click Test to verify
5. Start chatting

For local providers like Ollama and LM Studio, no API key is needed — just point the extension at your local server.

★ SUPPORT
Open an issue at https://github.com/alvescrafter/byok-ai-chat/issues
```
